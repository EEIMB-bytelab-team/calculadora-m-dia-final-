# Calculadora de Nota da Prova Final

Este projeto é um site que ajuda estudantes a descobrirem **quanto precisam tirar na prova final** para serem aprovados na disciplina.
A ideia é facilitar a vida de quem já está cansado de fazer conta no papel ou ficar na dúvida se ainda dá pra passar.

# Objetivo

Permitir que o aluno informe sua média anual e veja rapidamente:  

* Se já está aprovado
* Se precisa fazer prova final
* Qual **nota mínima** ele precisa tirar na final para passar

# Como funciona

O estudante informa:

* A média anual na disciplina 

Com base nesse dado, o sistema calcula automaticamente a nota mínima necessária na prova final.

# Funcionalidades

* Interface simples e fácil de usar
* Cálculo automático da nota necessária na final
* Feedback claro para o estudante
* Evita erros de cálculo manual

# Tecnologias utilizadas

* HTML
* CSS
* JavaScript

#  Como usar

1. Acesse o site
2. Preencha sua media anual no campo indicado
3. Clique em **Calcular**
4. Veja o resultado e saiba exatamente quanto precisa tirar na final

# Público-alvo

* Estudantes do ensino médio
* Qualquer pessoa que precise calcular nota de prova final

# Observação

Os cálculos seguem uma lógica padrão de médias. Sempre confirme se a regra usada bate com a da sua instituição de ensino.
